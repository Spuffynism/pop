Pop open a project, wherever it is.

```shell
pop config directory add /projects1
pop config directory add /projects2
pop config editor vim
pop open project # opens project using vim
```

## Setup

```shell
make build
echo "PATH=\$PATH:$(pwd)" >> ~/.zshrc
```

## Usage

See `pop --help`

## Configuration

See `pop config --help`
